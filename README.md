## Getting Started

### Prerequisites

- A modern web browser
- (Optional) A local HTTP server for development preview

### Running Locally

Clone the repository and open `index.html` in your browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Deployment

This site is automatically deployed via GitHub Pages on every push to the `main` branch. No additional CI/CD configuration is required.
